# Allbirds - Fit & Sizing Guide

## Sizing by Model

### Wool Runner
- **Fits**: True to size
- **Note**: Wool stretches slightly over time. If between sizes, size down.
- **Break-in period**: 3-5 wears

### Wool Runner Mizzle (Waterproof)
- **Fits**: Slightly snug (waterproof lining takes up space)
- **Note**: Size up 0.5 if wearing thick socks.

### Tree Dasher (Performance)
- **Fits**: True to athletic size
- **Note**: If between sizes, size up for running socks.

### Tree Breezer (Flats)
- **Fits**: Runs large
- **Note**: Size down 1 full size from your sneaker size.

### Tree Skipper (Casual)
- **Fits**: True to size
- **Note**: Can be worn with or without socks.

### Wool Piper (Lace-up)

- **Fits**: True to size
- **Note**: Adjustable laces for custom fit.

## Measuring Your Feet
1. Place paper against a wall.
2. Stand on the paper with your heel touching the wall.
3. Mark the longest toe.
4. Measure the distance in inches or cm.
5. Use our size chart at allbirds.com/size-chart.

## Width
- Allbirds run medium (D) width.
- Natural materials stretch to accommodate slightly wider feet.
- For wide widths: try Wool Runners or Tree Dashers (most accommodating).
