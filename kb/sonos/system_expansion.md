# Sonos - System Expansion Guide

## Creating a Stereo Pair
1. Place both speakers in the same room (6-10 feet apart for best stereo imaging).
2. In the Sonos app, go to Settings > System > Add Stereo Pair.
3. Select the two speakers and designate left/right channels.
4. One speaker becomes the primary (handles WiFi) and the other is the secondary.

## Adding Surrounds (Arc + Sub)
### Setting up Sonos Arc + Sub + Era 100/300 as Surrounds
1. Set up your Arc as the primary home theater speaker via HDMI eARC.
2. Place the Sub anywhere in the room (omnidirectional bass).
3. Add surround speakers behind the listening position.
4. In the Sonos app: Settings > System > Add Surrounds.
5. Run TruePlay room tuning for optimal sound.

## S1 vs S2 Systems

### S2 (Modern System)
- Supports Arc, Beam (Gen 2), Era 100/300, Sub (Gen 3), Move 2.
- Higher resolution audio, more features.
- All new products use S2.

### S1 (Legacy System)
- Supports Play:1, Play:3, Play:5 (Gen 1), Connect.
- Cannot be grouped with S2 devices.
- If you have mixed old and new devices, you can run S1 and S2 on the same network separately.

### Migrating from S1 to S2
1. Check compatibility on sonos.com/s2.
2. In the S1 app, go to Settings > System > Update to S2.
3. S2 is free for all compatible devices.
